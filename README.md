# Summit Edge Ski Tuning & Repair

Single-page marketing site for a local ski tuning and repair business. Built with plain HTML, CSS, and JavaScript for easy maintenance.

## Customize

- Update business name, phone, address, and hours in `index.html`.
- Replace the placeholder form behavior in `script.js` with a real booking or email service.
- Adjust colors and fonts in `styles.css`.

## Preview

Open `index.html` in a browser to view the site.
